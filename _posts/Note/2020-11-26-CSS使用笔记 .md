1. 边框属性 border
```css
/* 圆角半径 */
border-radius: 20rpx;
/* 边框样式 */
/*  dotted: 定义一个点线边框
    dashed: 定义一个虚线边框
    solid: 定义实线边框
    double: 定义两个边框
    groove: 定义3D沟槽边框
    ridge: 定义3D脊边框
    inset:定义一个3D的嵌入边框
    outset: 定义一个3D突出边框 */
border-style: solid;
/* 边框宽度 */
border-width: 2rpx;
/* 边框颜色 */
border-color:  rgb(206, 205, 205);
/* 边框-简写属性 */
/* border:5px solid red; */
```
2. 布局flex( display: flex)
```css
    /* 布局方式 */
    display: flex;
    /* 布局方向 */
    /* column 水平方向
       row 竖直方向 */
    flex-direction: column;
    /* 超出换行 */
    flex-wrap: wrap;
    /* 靠右对齐 */
    justify-content: flex-start;
    /* 靠上对齐 */
    align-items: flex-start;
```
    
3. 文本超框显示省略号
```css
    /* 内容超框后禁止换行显示 */
    white-space: nowrap;
    /* 超出部分隐藏 */
    overflow: hidden;
    /* 文字超出部分以省略号显示 */
    text-overflow: ellipsis;
```
