1. 边框属性 border
```css
/* 圆角半径 */
border-radius: 20rpx;
/* 边框样式 */
/*  dotted: 定义一个点线边框
    dashed: 定义一个虚线边框
    solid: 定义实线边框
    double: 定义两个边框
    groove: 定义3D沟槽边框
    ridge: 定义3D脊边框
    inset:定义一个3D的嵌入边框
    outset: 定义一个3D突出边框 */
border-style: solid;
/* 边框宽度 */
border-width: 2rpx;
/* 边框颜色 */
border-color:  rgb(206, 205, 205);
/* 边框-简写属性 */
/* border:5px solid red; */
```
